<!-- GETTING STARTED -->
## Getting Started

### Here's how to download and use DAX.



1. Clone the repo
   ```sh
   git clone https://github.com/sushiguy35/DAX.git
   ```
2. Run the build command
   ```sh
   g++ src/main.cpp -o dax.exe
   ```
3. Run the program
   ```sh
    ./dax.exe <your dax file>
   ```
